◆ __attribute__()

struct __attribute__ ( (packed)  )

A basic packet header. 0 1 2 6 8 (<= 509? 522?) +------------------------------------------------------—+ | type | rank | seqNo | blk | payload | +------------------------------------------------------—+ type bits: 0 1 4 5-7.

+----------------—+ |A|TYP|D| reserved | +----------------—+ A: ACK bit: currently (unfortunately) used to extend TYP in some cases TYP: Packet type D: "Decoded": obsolete reserved: used to be 12 bits for length

Fat ACK format, in addition to the header above: 8 12 14 16 ... +------------------------------------------------------------------—+ | cumDPRecv | leftBlock | # blocks | rank for block #leftBlock... | +------------------------------------------------------------------—+

Make a connection packet.

Copies payloadLength bytes of payload to buf, prepending an appropriate header based on blk and seqNo. Returns the used length of buf, which will be <= bufLength, or -1 on error. Appears to be exactly the same as a data packet, except for the flag bit in the header.
